How long does it take to download the page¶

S0 = size of base file (bits) N = number of inline objects S = size of inline object (bits) C = link of capacity (bits/s) D = RTT

  • HTTP/1.0 Non-persistent HTTP (without parallelism)

    • 2D + S0/C + N(2D + S/C)

  • HTTP/1.1 persistent HTTP (without parallelism or pipelining)

    • 2D + S0/C + N(D + S/C)

  • HTTP/1.1 persistent HTTP with pipelining

    • 3D + S0/C + NS/C

DNS records¶

  • distributed database storing resource records (RR)
  • RR format: (name, value, type, ttl)
  • type
    • A
      • name is hostname
      • value is IP address
    • NS
      • name is domain (e.g., foo.com)
      • value is hostname of authoritative name server for this domain
    • CNAME
      • name is alias name for some canonical (the real) name
      • www.ibm.com is really servereast.backup2.ibm.com
      • value is canonical name
    • MX
      • value is name of mailserver associated with name
